# Privacy Policy

## Data Collection

This extension stores barcode data locally on your device and optionally syncs to a cloud database (Supabase) for backup and cross-device access.

## Local Storage

- Barcode values and category names are stored locally using Chrome's storage API
- Login credentials (store number and password hash) are stored in session storage

## Cloud Sync (Optional)

When enabled, the following data is synced to Supabase:
- Store number (for account identification)
- Category names
- Barcode values
- No personal information beyond store identifiers

## Data Usage

- Barcode data is used solely for list management functionality
- No analytics or tracking
- No data sold or shared with third parties

## Data Deletion

To delete all stored data:
1. Uninstall the extension
2. Clear browser data for this extension

Cloud data can be deleted by contacting the extension developer.
